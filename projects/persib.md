# ⚽ Persib Fan Engagement Platform

A large-scale football application designed for fans to access real-time match updates, team information, and ticket purchasing.

---

## 🚀 Overview

The Persib app is a consumer-facing platform that provides live match updates, player details, and interactive fan engagement features.

---

## 🎯 Key Features

- 📅 Match schedules & fixtures
- 🔴 Real-time match updates (Socket.io)
- 👤 Player profiles with details
- 📰 Team news & updates
- 🎟️ Online ticket purchasing
- 📥 Downloadable fixtures

---

## 📊 Impact

- 📱 1M+ downloads within first month
- 🎟️ 3K–6K ticket sales
- 💰 Real-world payment integration

---

## 🧠 My Responsibilities

- Built real-time update system using WebSocket
- Developed CMS APIs using Strapi
- Integrated payment gateway for ticket sales
