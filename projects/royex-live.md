# 🎥 Royex Live

A real-time video conferencing platform supporting multi-user communication with chat and file sharing.

---

## 🚀 Overview

Built for internal team collaboration, this application enables users to create and join meetings with real-time video/audio communication.

---

## 🎯 Key Features

- 🎥 Video & audio calls (WebRTC)
- 👥 25–30 concurrent users
- 💬 In-call chat system
- 📎 File sharing
- 🎛️ Host controls (mute, kick, end call)

---

## 🧠 My Responsibilities

- Built full-stack application using Node.js & WebRTC
- Implemented signaling using Socket.io
- Ensured low-latency communication
- Developed UI and backend logic
